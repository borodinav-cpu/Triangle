Описание проекта

Консольное приложение на языке C#, которое определяет тип треугольника по заданным длинам его сторон и вычисляет координаты вершин для отображения в области размером 100x100 пикселей.
Функциональность

Программа выполняет следующие задачи:

Принимает три входных значения (длины сторон треугольника)
Проверяет корректность введённых данных
Определяет тип треугольника:
равносторонний
равнобедренный
разносторонний
не треугольник
Вычисляет координаты вершин треугольника
Масштабирует координаты для отображения в области 100x100 px
Выполняет логирование результатов
огирование

Программа ведёт логирование:

Успешных запросов (INFO)
Неуспешных запросов (ERROR)

Логи содержат:

Дату и время запроса
Входные параметры
Результат выполнения

Логирование осуществляется:

В консоль
В файл app.log
